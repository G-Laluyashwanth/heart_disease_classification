# Baseline Model Results

## Objective

Compare baseline machine learning models for heart disease prediction.

## Results

| Model | Accuracy |
|---------|---------|
| KNN | 0.6885 |
| Logistic Regression | 0.8852 |
| Random Forest | 0.8361 |

## Observations

- Logistic Regression achieved the highest baseline accuracy.
- Random Forest performed competitively.
- KNN underperformed compared to other models.
- Logistic Regression will be used as the initial benchmark model for further optimization.
